# CV — Federico Ressi

[View this file on GitHub (latest)](https://github.com/FedericoRessi/cv/blob/main/README.md)

This repository holds the **source of truth** for my curriculum vitae in a structured, machine-friendly format, plus a **human-readable copy** for sharing and printing.

## Contents

| File | Role |
|------|------|
| [CV.yaml](CV.yaml) | Single YAML document with contact info, summary, skills, work history, education, and interests. Easy to diff, script, or feed into other tools. |
| [CV.md](CV.md) | Markdown version generated from the same information, suitable for GitHub, viewers, or export to PDF/HTML. |

`CV.md` is maintained to match `CV.yaml`; if you change one, update the other.
